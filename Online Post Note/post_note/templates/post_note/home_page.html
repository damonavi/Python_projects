ONLINE POST IT NOTE

Online post it note note is a web application that allows a user to post different kinds of notes and thoughts on his personal dashboard. It requires for the user to register and login into their respective account and add, delete and update different posts. it also serves as a feature to view different posts and send email to their email account given at the time of registration. For loggin in, the user has to first register himself.

ABOUT THE PROJECT:

The whole project is developed by Abhishek Kumar Jha, a 3rd year Computer Science and Engineering student and the whole structure and organisation of project is uploaded to a github repository https://github.com/damonavi/webApps/postNote. The project make use of various web development components and uses python django framework for it.
